# NRC Project

This project is developed as part of a lab assignment.

## Description
This project includes basic geometric classes implemented in Python.

## Classes

### Point
Represents a point in 2D space.

### Line
Represents a line defined by two points.

## Features
- Distance calculation between two points
- Line length calculation
- Slope calculation

- Python
- Pytest
